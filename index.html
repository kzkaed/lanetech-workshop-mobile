We will explore components of xcode, including how to build and use the emulator

Getting Started

  1. Open up Xcode
  2. Create a new Xcode project
  3. IOS Single View Application
  4. Product Name ""
  5. Organization Name anything
  6. Organization Identifier unique to you
  7. Swift - Historically, development for IOS happened in Objective C. It has transitioned to Swift, and while xcode still supports both, we will only be working in Swift. All new development in xcode should be in Swift.
  8. Universal
  9. Pick a directory to save project

Navigating Xcode

Navigator - Left hand Side

Shows all project files in app Can search Look at old builds

Utilities - Right hand Side

The top part is an inspector, where you can analyse and change attributes The bottom is a palette of UI components
